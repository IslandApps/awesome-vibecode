## 🏗️ App Builders
> **One-shot platforms to go from prompt to production.**

* **[Base44](https://base44.com/)** — An AI-first "app factory" that generates full-stack logic and infrastructure on the fly. It automatically handles user logins, databases, and permissions behind the scenes.
* **[Bolt.new](https://bolt.new/)** — StackBlitz’s flagship prompt-to-app engine. It runs a full development environment in your browser, allowing you to install npm packages and deploy instantly.
* **[Lovable](https://lovable.dev/)** — The gold standard for "Aesthetic" web apps. It writes exceptionally clean code, manages GitHub sync, and allows for high-end software creation through chat.

## 🛠️ Editors
> **The primary canvases where your ideas take shape.**

* **[Google Antigravity IDE](https://antigravity.google)** — An agent-first development platform built on VS Code. Orchestrates autonomous agents that plan, code, and verify tasks across your editor and browser.
* **[Trae](https://www.trae.ai/)** — ByteDance’s "vibe-native" IDE fork of VS Code. Features **"SOLO Mode"** for autonomous task execution and **"CUE"** for predictive autocompletion.
* **[Visual Studio Code](https://code.visualstudio.com/)** — The extensible foundation of the modern dev stack. Its massive ecosystem of extensions makes it the essential canvas for vibe coding.
* **[VSCodium](https://vscodium.com/)** — The **telemetry-free**, community-driven binary distribution of VS Code. Exact same experience as VS Code but with all tracking stripped out.

## 🔌 Extensions
> **Force-multipliers and agents that live inside your IDE.**

* **[Augment Code](https://www.augmentcode.com/)** — A high-performance AI coding assistant built for massive codebases. Its **"Context Engine"** can process 400,000+ files with architectural awareness.
* **[Cline](https://cline.bot/)** — The most adopted open-source coding agent extension for VS Code. Executes multi-file edits, runs terminal commands, and debugs in real-time.
* **[Codex (OpenAI)](https://openai.com/codex)** — A standalone macOS "command center" for multi-agent orchestration. Features built-in **Git worktree** support for parallel tasking.
* **[ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)** — The essential "sanity check." Acts as a guardrail that catches agent hallucinations before they break your build.
* **[Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)** — An opinionated code formatter. Ensures that no matter which agent wrote the code, the project stays "vibe-consistent."
* **[Roo Code](https://github.com/RooCode/RooCode)** — A powerful, high-speed fork of Cline designed for maximum flexibility. It supports **Custom Modes** for specific task types.
* **[Tailwind CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss)** — The official extension for Tailwind. Provides instant autocomplete and linting for utility classes.
