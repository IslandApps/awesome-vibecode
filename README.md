## 🔊 Audio

* **[Incompetech (Kevin MacLeod)](https://incompetech.com/music/royalty-free/pieces.json)** — A direct JSON endpoint for a massive library of royalty-free background music. Great for programmatically adding atmospheric tracks to projects with easy attribution.

## 🛠️ Editors

* **[Google Antigravity IDE](https://antigravity.google)** — An agent-first development platform built on VS Code. It moves beyond autocomplete into "Mission Control," allowing you to orchestrate autonomous agents that plan, code, and verify tasks.
* **[Trae](https://www.trae.ai/)** — ByteDance’s "vibe-native" IDE fork of VS Code. Features **"SOLO Mode"** for autonomous task execution and **"CUE"** for predictive, multi-line autocompletion.
* **[Visual Studio Code](https://code.visualstudio.com/)** — The extensible foundation of the modern dev stack. Its massive ecosystem of extensions and AI forks makes it the essential canvas for vibe coding.

## 🚀 Generative Engines

* **[Augment Code](https://www.augmentcode.com/)** — A high-performance AI coding assistant built for massive codebases. Its **"Context Engine"** can process 400,000+ files with architectural awareness.
* **[Z.ai Chat](https://z.ai/)** — A powerful, **completely free** browser-based agent. It acts as a universal creation engine—simply prompt it to build full-stack web apps, responsive websites, professional slide decks, or deep research reports.

## 💻 Local AI

* **[LM Studio](https://lmstudio.ai/)** — The gold standard for running local LLMs with a polished GUI. Features parallel inference, a built-in local OpenAI-compatible server, and the **"llmster"** daemon for headless deployments. Supports GGUF and MLX engines, making it the perfect tool for zero-latency, private prototyping.

## 🧠 Models

* **[Claude (Anthropic)](https://www.anthropic.com/claude)** — The standard for "human-like" reasoning and elite coding. Features the **Claude 4.5/4.6** series. Its dual reasoning modes make it the most versatile model for complex architectural decisions.
* **[Gemini (Google)](https://ai.google.dev/)** — The multimodal king. **Gemini 3 Pro** features a massive 2M+ context window, allowing you to "vibe" with an entire codebase at once.
* **[GLM Coding Plan (z.ai)](https://z.ai/subscribe)** — **The "Budget King" of coding plans.** For as little as **$3/month**, you get a high-performance GLM-5 model with roughly **3x the usage limits** of a standard $20 subscription.
* **[xAI (Grok)](https://x.ai/)** — Built for raw speed and high volume. **Grok 4.1 Fast** is priced aggressively ($0.20 per 1M tokens), making it the ultimate engine for high-output projects and real-time data integration.
